## **조건문**

<br/>

**[1] if 조건문**


```
if a<100:
    print(a)
```

<br/>

**[2] if .. else조건문**

```
if a<100:
    print("a<100")
else:
    print("a>100")
```

<br/>

**[3] if .. elif .. else조건문**

```
if a<100:
    print("a < 100")
elif a = 0:
    print("a = 0")
else:
    print("a > 100")

```
=>  else문 안쓰고 elif만 써도 에러는 안나옴
