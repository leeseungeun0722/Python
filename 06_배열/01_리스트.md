## **리스트란 무엇인가**

<br/>

**[1] 리스트란?**

=> 하나의 리스트 안에 다양한 데이터 값을 담을 수 있는 바구니, 효율성 높아진


<br/>

**[2] 리스트**
<br/>

=>리스트 요소로는 [10,3.14,'a','Korea']등과 같이 정수형, 실수형, 문자형을 하나의 리스트에 담을 수 있다.
  
=> 리스트 안에 또 다른 리스트 요소로써 가질 수 있다

=> 중복된 값을 가질 수 있다.

=> 빈 리스트를 생성하는 방법은 **a = [ ]** 방식과 **b = list ()**방식중 어느것을 사용해도 무방
